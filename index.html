## Descrição e objetivos ### Descrição - Este projeto está sendo desenvolvido para um projeto de **TCC** **UFERSA** - Universidade Ferderal Rural do Semi-Arido - Nesse repositório ficará o codigo da parte de servidor e pagina web O codigo completo com todas as funcionalidades ficaram em outro repositório ### Objetivos - Criar um servidor web que rode em uma placa NodeMCU/Esp8266. - Criar uma pagina Web com campos para o usuário inserir as medidas do reservatório. - A pagina web deve ter um campo de seleção onde se possa escolher o formato do reservatório. - Os campos de dados dever ser exibidos/olcutos de acordo com o reservatório escolhido pelo usuário. - Os dados inseridos devem ser enviados para o servidor no ESP8266 para serem tratados ### Instruções - O arquivo pagina.h é o mesmo que o pagina.html, podendo qualquer um ser renomeado para edição ou para uso. - Para usar, o arquivo deve ter a extenção .h, para editar recomendo renomear para exteção .html e comentar a primeira e ultima linha do código. Atenção: quando vor renomear novamente para .h lembre de descomentar a primeira e última linha que foram comentadas anteriormente
